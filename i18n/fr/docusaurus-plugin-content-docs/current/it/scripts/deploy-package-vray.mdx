# Présentation

Ce script copie simplement le contenu du plugin v-ray présent sur le 192.168.2.112 sur la machine locale.
Le lien entre le dossier sur la machine locale et Houdini est effectué grace au fichier .json a placé dans le dossier 'package' là où est installé Houdini.
